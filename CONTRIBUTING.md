# Contributing to Nearchat

Thank you for your interest in contributing to **Nearchat** — an open‑source, ad‑free social app designed to connect local communities.

This project is at an early stage, and contributors can help shape the foundation of the app.  
Everyone is welcome: developers, designers, testers, writers, and community members.

---

## 🚀 Project Vision

Nearchat aims to create a modern, privacy‑friendly platform where people can connect with others living nearby.  
It is NOT a dating app — it is a community tool focused on communication, collaboration, and local interaction.

---

## 🧱 Current Status

The project is in the **planning and documentation** phase.  
There is no production code yet.  
We are building:

- the feature list  
- the UI/UX structure  
- the technical roadmap  
- the open‑source community  

Your ideas and contributions are highly valuable at this stage.

---

## 🛠 How to Contribute

### 1. Development
We welcome contributions in:

- Flutter  
- React Native  
- Kotlin / Swift  
- Node.js / Supabase / Firebase  
- Mapbox / OpenStreetMap  

You can help by:
- implementing features from the roadmap  
- proposing architecture  
- fixing issues  
- improving performance  
- building UI components  

---

### 2. UI/UX Design

---

### 3. Documentation
You can contribute by:
- improving README  
- writing technical docs

- organizing the roadmap  

---

### 4. Community & Ideas
You can:
- open Issues with feature proposals  
- discuss improvements  
- help define priorities  
- test early builds  
- give feedback  

---

## 📌 Contribution Workflow

1. **Fork** the repository  
2. Create a new branch:  
   `git checkout -b feature-name`  
3. Make your changes  
4. Commit with a clear message:  
   `git commit -m "Add: implemented user profile screen"`  
5. Push the branch:  
   `git push origin feature-name`  
6. Open a **Pull Request**  
7. Participate in review and discussion  

---

## 📜 Code of Conduct

Be respectful, constructive, and supportive.  
We are building a community app — the community starts here.

Harassment, discrimination, or toxic behavior will not be tolerated.
