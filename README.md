# Nearchat

Nearchat is an open‑source, ad‑free social app designed to connect people within the same local community.  
It helps neighbors discover each other, communicate safely, and build stronger local networks.

This project is fully open‑source, community‑driven, and built in public.

---

## 🌍 What is Nearchat?

Nearchat is a location‑based social platform focused on local communities.  
It is NOT a dating app.  
It is a tool for communication, collaboration, and interaction between people who live near each other.

Users can:
- See nearby users (approximate location)
- Join local groups (building, staircase, neighborhood)
- Post in a local feed
- Send private messages after mutual acceptance
- Share announcements (lost & found, help, buy/sell)
- Control visibility with Ghost Mode

---

## 🧩 Core Features (MVP)

### • Community Map
Shows nearby users with approximate location.  
Visibility can be toggled anytime.

### • User Profiles
Photo, nickname, short bio, interests.  
SMS verification to reduce fake accounts.

### • Private Chat
Only after mutual acceptance.  
Text, images, reactions.  
Block/report options.

### • Local Feed
Posts visible only in the local area.  
Likes and comments.

### • Local Groups
Building group  
Staircase group  
Neighborhood group  
Thematic groups

### • Announcements
Lost & found  
Help requests  
Buy/sell  
Recommendations

### • Privacy Controls
Ghost Mode  
Visibility toggle  
Notification settings
